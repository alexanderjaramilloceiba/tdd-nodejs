# pruebas unitarias

## express + jest

crear un proyecto desde cero

    npm init -y
    npm i express --save

    npm i typescript ts-node nodemon -D
    npm i @types/node @types/express -D

    npm run tsc -- --init

librería de pruebas jest

    npm i jest ts-jest -D
    npm i @types/jest -D

    npm run ts-jest config:init
    # o #npx ts-jest config:init

## iniciar proyecto

    npm i
    npm start
